# Проект: Анализ поведения пользователей для повышения удержания клиентов

## Описание
Исследовательский анализ данных о заказах пользователей платформы с целью выявления закономерностей в поведении и определения факторов, влияющих на вероятность возврата клиентов.

## Цель
Выявить пользователей с высокой вероятностью возврата и сформулировать рекомендации для оптимизации маркетинговых бюджетов.

## Структура проекта
- `Project_Module_2.ipynb` — основной ноутбук с анализом
- `requirements.txt` — зависимости проекта
- `.env` — переменные окружения (не загружается в репозиторий)

## Установка
```bash
pip install -r requirements.txt
